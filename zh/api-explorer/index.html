    概述

    您可以通过调用API使用所需的AI功能。

    • 轻量级文字识别（简体中文）
    • 轻量级文字识别（繁体中文）
    • 高阶文字识别（多语种）
    • 自定义模板文字识别
    • 车牌信息识别
    • 人脸检测
    • 人脸相似度比对
    • 人体结构化属性
    • 智能人像分割
    • 图像相似度
    • 通用物体识别
    • 色情内容检测
    • 图像超分辨率
    • 文本相似度
    • 通用自然语言理解

    调用方式

    调用AI功能的URL有两种方式：

    • 如果您在部署解决方案时，将参数APIGatewayAuthorization设置为NONE，API将允许匿名访问。
    • 如果您在部署解决方案时，将参数APIGatewayAuthorization设置为AWS_IAM，将被要求使用AWS IAM身份验证的调用API。详情可参见在Amazon API Gateway中调用REST API。
